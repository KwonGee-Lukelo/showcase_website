# Burger Guys – Site vitrine d'un restaurant de burgers

## À propos du projet

**Burger Guys** est un site vitrine moderne, responsive et accessible conçu pour mettre en valeur un restaurant spécialisé dans les burgers. Le projet présente l'univers du restaurant, son menu, ses spécialités ainsi que ses informations de contact à travers une interface soignée et une navigation intuitive.

Ce projet a été réalisé dans le cadre de la **Phase 1 – HTML, CSS & Git** de **AKIENI Academy** afin de consolider les compétences fondamentales en développement Front-End et en conception d'interfaces web modernes.

## Responsive Design

Le site a été développé selon une approche **Mobile First** afin d'offrir une expérience utilisateur optimale sur tous les types d'écrans :

- Smartphone
- Tablette
- Ordinateur

---

## Points forts du design

- Interface moderne et élégante
- Palette de couleurs inspirée de l'univers de la restauration
- Navigation simple et intuitive
- Hiérarchie visuelle claire
- Typographie moderne
- Effets de survol et animations CSS subtiles
- Mise en page responsive

---

## Accessibilité

Le projet applique plusieurs bonnes pratiques en matière d'accessibilité :

- HTML5 sémantique
- Attributs `alt` sur les images
- Hiérarchie correcte des titres
- Contrastes de couleurs adaptés
- Navigation au clavier

## Installation

Clonez le dépôt :

```bash
git clone https://github.com/KwonGee-Lukelo/showcase_website.git
```

Accédez au dossier du projet :

```bash
cd showcase_website
```

Puis ouvrez le fichier **index.html** dans votre navigateur.
